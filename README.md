## 🎮 Minecraft Bedrock development

- **Gameplay scripting** on the Bedrock Script API (`@minecraft/server`) in TypeScript and JavaScript.
- **Custom mob and boss AI**: taming, attacks, abilities, companion detection.
- **World generation**: features, feature rules and procedural dungeon/structure generation.
- **Custom content**: blocks, items, loot tables and recipes through reusable behavior-pack components.
- **Clean, event-driven architecture** designed to scale across large add-ons.

## 💻 Software, plugins and bots

- **Plugin development** for game servers and platforms (Java and more).
- **Bot development**: automation and community bots.
- **Software development** in C, C++, Java, PHP and TypeScript.
- **Web design and development**: front end to back end, from layout to server-side logic.

## 🖥️ Sysadmin and infrastructure

- **Linux system administration**: server setup, hardening, maintenance and monitoring.
- **Reverse proxy with NGINX**: routing, load balancing, SSL termination.
- **Deployment and self-hosting** of web apps, game servers and services.
